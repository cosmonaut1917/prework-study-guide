HTML

  • The head element contains information about the webpage.
  • The body element contains the visible content on the webpage.

CSS

  • using a style element in the head element can allow you to embed an entire stylesheet.
  • use a style element after the head element followed by your h1 and color then close your style element.
  • when using an external style sheet you link the file using a link element placed in the html head.
  • A margin indicaties how much space we want around the outside of the element
  • A padding indicates how much space we want around the inside of the element.

GIT

  • Git status checks the status of the branch
  • Git checkout -b branch name makes a new branch off of the main branch.
  • Git merge pulls all side branches back into the main branch.
  • Git commit pushes the code to github.

JAVASCRIPT

  • the script . js element is linked in the body element
